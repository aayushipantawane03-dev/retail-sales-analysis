# retail-sales-analysis
Mini project on retail sales data analysis using Python and Pandas.

# Objectives
- Load and inspect raw retail sales data  
- Clean missing values and fix incorrect data types  
- Engineer time-based features (Year, Month)  
- Perform group-by aggregations to generate business insights
  
# Tools & Libraries Used
- Python  
- Pandas  
- Jupyter Notebook

# Key Analysis Performed
- Year-wise total sales  
- Category-wise total sales  
- Region-wise and city-wise total sales  
- Monthly sales trends  
- Top 10 products by total sales
